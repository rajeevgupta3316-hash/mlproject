# CAD and Diabetes Disease Prediction

## Project Description
This project aims to predict CAD (Coronary Artery Disease) and diabetes using Machine Learning techniques, particularly Support Vector Machines (SVM) and Artificial Neural Networks (ANN). The project will involve loading, preprocessing, training, and evaluating models to achieve accurate predictions.

## Objectives
- Implement a data preprocessing pipeline.
- Develop and train SVM and ANN models.
- Evaluate model performance using appropriate metrics.
- Provide user-friendly instructions for model usage.

## Models
1. **Support Vector Machine (SVM)** - A classic machine learning algorithm for classification tasks.
2. **Artificial Neural Network (ANN)** - A more complex model suited for capturing non-linear relationships in data.

## Usage Instructions
1. Install the required dependencies using `pip install -r requirements.txt`.
2. Use `python train.py` to train the models.
3. Make predictions using `python predict.py`.
4. Evaluate models with `python evaluation.py`.
